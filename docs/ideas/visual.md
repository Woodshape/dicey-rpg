# Visual Ideas

## Dice Type Shapes

- Currently dice are represented by plain colours. Adding distinct shapes would make each type recognisable without relying on colour alone (accessibility, at-a-glance clarity).
- Options:
  - Polygon matching face count: d4 = triangle, d6 = square, d8 = diamond, d10 = pentagon, d12 = hexagon
  - Die type label (d4, d6, etc.) rendered inside the coloured shape
  - Symbolic icons (e.g., a small numeral for the face count)
- Skull dice have a distinct bone-white colour already — adding a symbol (skull icon or crossbones) would reinforce the distinction further.
